IrfanView Toolbar Skins

How to install additional toolbar skins for IrfanView 4.00 or later?

  • Download toolbar ZIP file (click on the toolbar image below)
  • Unzip the downloaded file into your "IrfanView\Toolbars\" folder
  • Start IrfanView and go to "Properties->Toolbars" to change the skin

A huge THANKS to all users sending me icons and toolbars during last years


Designers: if you create a new toolbar skin, please send me an email with the PNG/TXT files. THANKS!
